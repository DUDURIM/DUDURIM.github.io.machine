방명록 서비스

경조사 방명록을 디지털화

분류: 개인

진행 기간: 2022.10.17 ~ 2022.10.21

Skills: Visual Studio Code, HTML, CSS, Javascript, jQuery, Node.js, MySQL

팀 구성: 정구영

서비스 내용

결혼식장이나 장례식장에서 방명록을 아직까지 수기로 작성하는 것을 보고, PC나 모바일로 간편하게 기록할 수 있도록 만든 미니 프로젝트

성장 경험

1. 회원가입, 로그인 개념 파악
Node.js - Mysql와 연동하여 회원가입, 로그인, 수정, 탈퇴 기능 (sequelize 사용)

👀 서비스 화면

주요 기능

- 회원가입
-> 입력한 정보를 Sequelize create 메소드를 사용하여 DB에 INSERT

- 로그인
-> 로그인 버튼 클릭 시 Axios 요청을 하여 입력된 정보가 DB에 존재하는지 확인 (findOne 메소드 사용)
-> 로그인 성공 시 session 유저정보 저장

- 수정
-> findOne 메소드를 이용해 DB에서 로그인된 유저의 데이터를 찾아와 update 메소드로 수정 기능을 구현

- 탈퇴
-> destroy 메소드로 삭제 기능을 구현


- 계정별로 분리된 개인 방명록 구현
-> session의 user 정보를 사용해서 나의 개인 방명록 구현
(findAll 메소드 사용)
-> update 메소드로 수정 기능을 구현
-> destroy 메소드로 삭제 기능을 구현
