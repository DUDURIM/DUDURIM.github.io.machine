RANKING SYSTEM USING QUIZ

기업, 친구를 상대로 사용하는 랭킹 이벤트 시스템

분류: 개인

진행 기간: 2022.04.01

Skills: Visual Studio Code, HTML, CSS, Javascript, jQuery, Bootstrap, React.js, Firebase, AWS

팀 구성: 정구영

서비스 내용

이 페이지는 퀴즈를 통해 서로의 순위를 매겨볼 수 있다.
React를 사용하였고 퀴즈를 이용한 랭킹 시스템으로,
Firebase로 데이터가 연동 되어있어서 점수와 응원 메시지 기록이 계속 남아있으며 도메인 구매를 통하여 AWS로 서비스도 해보았다.
(AWS 과금 문제로 현재는 중단한 상태입니다. )

성장 경험

1. 데이터베이스 연동
Firebase를 이용하여 데이터를 연동해서 서비스를 해보았다.

2. 과금에 대한 경각심
AWS를 생각 없이 사용하다 과금 사태를 겪고 무슨 일이든 신중하게 접근하는 습관이 생겼다.

👀 서비스 화면

주요 기능
1. styled-components을 적용하여 스타일 구현
2. route로 페이지별 이동 구현
3. ProgressBar 애니메이션 구현
4. 머테리얼 UI 아이콘을 사용해서 로딩 스피너 구현
5. firebase 데이터 연동
