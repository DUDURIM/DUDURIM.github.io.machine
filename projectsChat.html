채팅 서비스

채팅 기능 심화연습

분류: 개인

진행 기간: 2022.09.16 ~ 2022.10.07

Skills: Visual Studio Code, HTML, CSS, Javascript, jQuery, Node.js

팀 구성: 정구영

서비스 내용

채팅 기능을 좀 더 깊게 공부하기 위해 채팅만 따로 분리하여 만든 미니 프로젝트

성장 경험

1. 소켓 개념 파악
소켓을 이용하면서 발생하는 오류들을 해결하며 개념을 파악할 수 있었다.

👀 서비스 화면

주요 기능 (socket.io 활용 및 jQuery 함수 사용)
1. 채팅방 입장 시 상단 INPUT, BUTTON 비활성화
2. 오토 스크롤로 채팅 입력 시 부드럽게 내려가게 구현
3. 채팅방 참가자 DM 기능 구현
4. 채팅 INPUT에 내용이 없을 시 전송 버튼 비활성화
5. 참가자 입장 메시지, 퇴장 메시지 기능
