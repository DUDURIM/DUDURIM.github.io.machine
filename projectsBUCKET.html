나의 버킷리스트 기록용 프로젝트

분류: 개인

진행 기간: 2022.03.01

Skills: Visual Studio Code, HTML, CSS, Javascript, jQuery, Bootstrap, React.js, Firebase, AWS

팀 구성: 정구영

서비스 내용

나만의 버킷리스트를 기록하고 싶어 만들게 된 페이지로 버킷리스트를 기록하는데 필요한 필수 요소들만 모아서 한눈에 보기 편하게 되어있다.

React를 사용하였으며 Firebase로 데이터가 연동 되어있어 원할 때 데이터의 기록, 삭제가 모두 가능하다.

성장 경험

1. 데이터베이스 연동
Firebase를 이용하여 데이터를 연동해서 서비스를 해보았다.

주요 기능
1. styled-components을 적용하여 스타일 구현
2. route로 페이지별 이동 구현
3. ProgressBar 애니메이션 구현
4. 머테리얼 UI 아이콘을 사용해서 로딩 스피너 구현
5. firebase 데이터 연동
